Linuxコマンド集

コマンド 説明 使用例
ls ファイル一覧表示 ls -l
cd ディレクトリ移動 cd LinuxStudySite
cp ファイルをコピー cp test.txt backup.txt
mv 移動・名前変更 mv test.txt test2.txt
grep 文字列検索 grep "収入" kakeibo.txt
awk 列の抽出 awk '{print $1}' kakeibo.txt
head 先頭行を表示 head -n 5 file.txt
tail 末尾行を表示 tail -n 5 file.txt
sort 並び替え sort 2023-09_log.txt
find ファイル検索 find . -name "*.txt"
date 日時を表示 date
free メモリ使用量確認 free -h
cat ファイル内容表示 cat kakeibo.txt
mkdir ディレクトリ作成 mkdir Test
touch 空ファイル作成 touch test.txt
pwd 現在地確認 pwd
rm ファイルを削除 rm test.txt
tree ディレクトリ構造を表示 tree
cut 列を抽出する cut -f 1 -d " " file.txt
wc 行数や文字数を集計 wc -l file.txt
echo 文字列を出力 echo Hello
less ファイル内容を閲覧 less log.txt
nano ファイルを編集 nano test.txt
chmod アクセス権を変更 chmod 755 test.sh
sudo 管理者権限で実行 sudo apt install tree
history 実行したコマンド履歴を表示 history
